# TimeTracker - Frontend

A minimal time tracking web application that helps you monitor work sessions.

## Features

- **Start/Stop Timer**: Track your work sessions
- **Status**: View current timer state and elapsed time
- **Total**: See cumulative work time across all sessions
- **Reset**: Clear timer and total time

## Backend

This frontend connects to the [TimeTracker API](https://github.com/Umutalb/TimeTracker-API).

## Usage

1. Click **Start** to begin tracking
2. Click **Stop** to end the session
3. Use **Status** to check current state
4. Use **Total** to see total work time
5. Use **Reset** to clear everything

## Tech Stack

- Vanilla JavaScript
- HTML5 & CSS3
- REST API integration
